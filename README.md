# PPPick

## 프로젝트 소개
- PersonalPicturePick의 약자로, 회원이 작가의 포트폴리오를 확인 후, 작가를 선택(Pick)하여 개인 맞춤형 일러스트/그림/디자인을 제작 의뢰하는 서비스입니다.
-
